# Rename

Rename or cast datatypes of columns

# columns

  • Type: array

  • Description: List of columns along with new names. columns not specified will remain as is. Use output_schema to only select a subset of the columns

  • Examples:

- name: COUNTRY_NAME
  rename: ctry_nm
